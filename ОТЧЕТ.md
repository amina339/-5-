4. Вначале обозначила оболочку Bash, далее создала переменную файла и проверила его формат. Если файл пустой то выводится сообщение в консоль и коммит не выполняется. Если файл не пустой, то тоже выводится сообщение и выполняется коммит. А если файл другого формата, то выводится сообщение и выполняется коммит.
```
    #!/bin/bash
   file=$(git diff --cached --name-only)
   if [[ "$file" == *.txt ]; then
       if [ ! -s "$file" ]; then
           echo "файл пустой, добавьте в него текст"
           exit 1
       else
           echo "Все файлы соответствуют требованиям"
           exit 0
       fi
   else
       echo "Нет файлов формата тхт, все ок"
       exit 0
   fi
```
